# Codex prompts

These prompts are designed to be copied into Codex with minimal editing. Replace placeholders such as `YOUR_ORG`, paths, and branch names.

Recommended usage:

1. Start each milestone by asking Codex to use the corresponding `P*.md` prompt in plan/audit mode.
2. After approving the plan, ask Codex to implement the approved plan in the same thread.
3. End each session with `R_update_docs_worklog.md` and then `R_review_diff.md`.
